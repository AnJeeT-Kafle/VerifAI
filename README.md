## What is VerifAI?

VerifAI is a Chrome Extension that helps users identify whether images on any webpage were generated by AI or taken by a real photographer. As AI-generated images become increasingly indistinguishable from real photographs, tools like VerifAI are critical for combating misinformation and synthetic media proliferation.

---

## Current Version (v2) — Per-Image Detection

Toggle detection mode on — a **Detect** button appears on every image on the page. Click any image to instantly get a verdict.

```
Toggle ON  →  Detect button appears on every image
                        ↓
           Click any image button
                        ↓
        ⚠ AI-Generated   or   ✔ Authentic
```

---

## Planned Version (v3) — Auto Detection

Once the self-hosted AWS Lambda server is complete, the extension will automatically scan every image on page load — no clicking required.

```
Toggle ON  →  All images scanned automatically
                        ↓
           ⏳ Scanning… appears on every image
                        ↓
        ⚠ AI-Generated   or   ✔ Authentic
           appears on all images at once
```

## Features

### Current (v2)
- 🔍 **Per-image detection** — click any image individually to check it
- ⚡ **Real-time results** — verdict appears in seconds
- 🎯 **Smart filtering** — ignores tiny icons and tracking pixels under 100×100px
- 🔄 **Infinite scroll support** — MutationObserver detects newly loaded images
- 🎨 **Visual overlays** — red border for AI-generated, green for authentic
- 💾 **Persistent toggle** — detection mode remembers state across sessions
- 🔒 **Secure key storage** — API keys stored in Chrome local storage

### Planned (v3)
- ⚡ **Auto scan** — all images scanned automatically on toggle
- 🖥️ **Self-hosted model** — own AWS Lambda server, no third-party API costs
- 📊 **Confidence score on badge** — shows percentage alongside verdict
- 🖱️ **Right-click detection** — check any single image via context menu
- 🌐 **Chrome Web Store** — public release for all users

## How It Works

### Current Architecture (v2)
```
User clicks Detect button
            ↓
content.js sends image URL to background.js
            ↓
background.js calls AI or Not API
            ↓
Returns verdict + confidence score
            ↓
Badge and border appear on image
```

### Planned Architecture (v3)
```
User toggles ON
            ↓
content.js auto scans all images
            ↓
background.js calls self-hosted AWS Lambda
            ↓
Lambda runs Hugging Face transformer model
            ↓
Returns { isAI, confidence, verdict }
            ↓
Badges appear on all images automatically
```

## API Research & Benchmarking

Multiple AI detection APIs were researched and evaluated before building a self-hosted solution:

| API | Verdict Format | Integration | Verdict |
|---|---|---|---|
| Reality Defender | deepfake / real | 3-step async — presign → upload → poll | Too complex for extension |
| Hive Moderation | probability score | Image upload, V2/V3 endpoints | AI detection locked behind paid tier |
| AI or Not | ai / human | Single POST with image URL | Best fit — simple and instant |

**Key Finding:** AI or Not offered the cleanest integration for a Chrome Extension — one API call, instant result. However, free tier credits are limited, which motivated the decision to build a self-hosted inference server.

## Installation (Local Development)

### Step 1 — Clone the repo
```bash
git clone https://github.com/CoLaXishere/verifai.git
cd verifai
```

### Step 2 — Load in Chrome
```
1. Open Chrome → chrome://extensions
2. Enable Developer Mode (top right toggle)
3. Click "Load unpacked"
4. Select the verifai/ folder
```

### Step 3 — Add your API key
Open the Service Worker console from `chrome://extensions` and run:
```js
chrome.storage.local.set({ aiOrNotKey: 'YOUR_API_KEY_HERE' })
```
Get a free API key at [aiornot.com](https://aiornot.com)

### Step 4 — Use it
```
1. Go to any webpage with images
2. Click the VerifAI icon in your toolbar
3. Toggle the switch ON
4. Click Detect on any image
```

## Research Motivation

This project was built to explore a real and growing research problem — as AI image generators (Midjourney, Stable Diffusion, DALL-E) produce increasingly photorealistic images, reliable detection is critical for:

- Combating misinformation in news and social media
- Protecting intellectual property of human artists
- Maintaining trust in visual media online

The next phase focuses on deploying a self-hosted Hugging Face transformer model on AWS Lambda to move beyond third-party APIs, enabling full control over the inference pipeline and unlimited detections at no cost.
